# OISRS
/*How to run the Online Information System for Client-Technician Interaction in provision of Refrigeration Services*/

1. Click the zipped file.
2. Extract the file, copy OISRS folder, and paste it on the desktop.
3. Copy paste the project folder in the htdocs folder, which located in the XAMPP Folder.
4. Activate XAMPP, and start Apache and MySQL.
5. Open MySQL Create a database techpythondb then import the SQL File available in the Database File Folder located within the OISRS folder.
6. Open Visual studio code and Click file on the top left, Tap open folder and open the OISRS Folder using VS Code.
7. Click on the terminal, and change to cmd terminal.
8. Navigate the project folder using the cd command.
9. Activate the virtual environment, Using the command source venv/Scripts/activate.bat while in that directory, will activate that virtual environment.
10. Run the Project using the following command: python manage.py runserver.
11. Copy the link and paste it the browser to start interacting with the OISRS.
